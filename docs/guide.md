# 项目使用与内容发布指南

本指南旨在帮助您了解如何在这个基于 Hugo 和 Stack 主题的博客项目中发布文章、管理分类与标签。

---

## 1. 如何发表文章

在本项目中发表文章非常简单，主要分为以下几个步骤：

1.  **创建文章目录**：在 `content/post/` 目录下创建一个新的文件夹（建议以文章的英文名或拼音命名）。
2.  **创建 index.md**：在新建的文件夹内创建 `index.md` 文件。
3.  **编写 Front Matter**：在文件开头编写 YAML 格式的文章元数据（标题、日期等）。
4.  **编写内容**：在元数据下方使用标准 Markdown 语法编写文章。
5.  **添加图片**：如果文章有封面图或插图，直接将图片文件放在该文章文件夹内，并在 Markdown 中引用。

---

## 2. 文章模板

您可以直接复制以下内容作为新文章的基础模板：

```markdown
---
title: "这里是文章标题"
description: "这里是文章的简短描述"
slug: "my-new-post"
date: 2026-03-09T08:51:00+08:00
image: "cover.jpg"
categories:
    - 技术
tags:
    - Hugo
    - Markdown
weight: 1
---

在这里开始编写您的正文内容...

## 第一章节
内容详情...

## 第二章节
内容详情...
```

## 3. 分类、标签与封面图管理

### 新建分类 (Categories)
1.  **在文章中使用**：直接在文章的 `categories` 字段下填写分类名称。
2.  **自定义分类页面**（可选）：
    - 在 `content/categories/` 下创建与分类同名的文件夹。
    - 在该文件夹内创建 `_index.md`。
    - 您可以为分类设置标题、描述、背景颜色等。

### 新建标签 (Tags)
- 在文章的 `tags` 字段下直接添加标签名称即可。Hugo 会自动为您汇总所有标签。

### 设置封面图 (Cover Image)
1.  将图片文件（如 `cover.jpg`）存放在文章所在的文件夹内。
2.  在文章的 Front Matter 中设置 `image: "cover.jpg"`。
3.  建议使用高质量的横向图片作为封面，以获得最佳视觉效果。

---

## 4. 关键参数解释

在文章开头的 Front Matter 中，以下字段非常重要：

- **slug**: 文章的 URL 别名。
    - *作用*：决定了文章在浏览器中的访问路径。如果不填，通常会使用文件夹名或标题。例如 `slug: my-post`，访问路径通常是 `/p/my-post/`。
- **date**: 文章的发布日期。
    - *作用*：用于文章排序（默认按日期倒序）以及在页面上显示发布时间。格式通常为 `YYYY-MM-DD`。
- **image**: 文章的封面图文件名。
    - *作用*：指定在文章列表页和文章详情页顶部显示的特色图片。只需填写放在同一目录下的图片文件名。
