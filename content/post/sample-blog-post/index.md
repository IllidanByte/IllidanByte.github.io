欢迎来到我的博客！这是通过 Hugo 自动生成的示例文章。

## 为什么选择 Hugo？

Hugo 是世界上最快的静态网站生成器。它的构建速度极快，且拥有丰富的模板生态系统。

### Stack 主题的优势

Stack 主题（hugo-theme-stack）由 [CaiJimmy](https://github.com/CaiJimmy) 开发，具有以下特点：

- **极简主义设计**：侧边栏导航，阅读体验极佳。
- **强大的卡片式布局**：文章列表以卡片形式展示，清晰美观。
- **完善的搜索功能**：内置基于 Fuse.js 的本地搜索。
- **暗色模式支持**：自动适配系统设置。

## 如何编写文章？

你可以使用标准的 Markdown 语法。例如，插入代码块：

```bash
hugo new post/my-new-post/index.md
```

或者添加引用：

> “简单是最高级的复杂。” —— 达·芬奇
